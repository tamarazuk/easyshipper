Easyshipper for WooCommerce allows your users to interface with the fantastic EasyPost Shipping API. 

== Description ==

EasyPost is a simple shipping API for your app or site. We believe in making purchasing labels, tracking packages, checking rates, verifying addresses, and all other shipping related functions as easy as possible.

== Installation ==

1. Install the EasyPoster - WooCommerce Easypost Plugin 
2. Activate the plugin
3. Go to the WooCommerce Settings Page 
4. Access Shipping Tab
5. Select "EasyPost"
6. Check the Enable/Disable Checkbox.
7. Enter the settings that you would like you to use
8. Enter the dimensions of the products, (Easypost uses this to determine suitability of products)
